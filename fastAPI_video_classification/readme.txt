Клиент-серверное приложение для классификации изображения с вебкамеры на основе FastAPI  -
клиент передает изображение с веб камеры на сервер,
сервер с помощью нейросети YOLO V5 предсказывает для этих изображений маску и 
отправляет обратно клиенту. 
Обмен между клиентом и сервером осуществляется либо через post-запрос, либо через вебсокеты

install python-multipart
uvicorn server:app --host 0.0.0.0 --port 12000 --reload
server - server.py
app - FasAPI application name in server.py
